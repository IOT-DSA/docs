LocalNode getOrCreateNode(
String fullPath,
[bool addToTree = true]
)

Gets a node at the given path if it exists. If it does not exist, create a new node and return it.

When addToTree is false, the node will not be inserted into the node provider.

Source

LocalNode getOrCreateNode(String fullPath, [bool addToTree = true]) {
  if (addToTree == true) {
    throw 'not supported';
  }

  String rPath = fullPath.replaceFirst(path, '');
  if (rPath == '') {
    rPath = '/';
  }
  RemoteLinkNode node = nodes[rPath];
  if (node == null) {
    node = new RemoteLinkNode(fullPath, broker, rPath, this);
    nodes[rPath] = node;
  }
  return node;
}
