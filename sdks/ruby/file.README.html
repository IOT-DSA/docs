DSLink SDK for Ruby

DSLink SDK for Ruby

Getting Started

Prerequisites

  • Git

  • Ruby

Install

Add the following to your Gemfile:

gem 'dslink', :git => 'git://github.com/IOT-DSA/sdk-dslink-ruby.git'

Usage

# link.rb
require 'dslink'
link = DSLink::Link.instance
link.provider.load({
    'hello-world' => {
        '$name' => 'Hello',
        '$type' => 'string',
        '?value' => ''
    }
})
link.connect do |success|
  link.provider.get_node('/hello-world').value = 'World'
end

Run: bash ruby link.rb --broker http://localhost:8080/conn --log debug
